# 📚 4. Sınıf Soru Bankası

MEB müfredatına uygun, interaktif 4. sınıf çalışma testi ve soru bankası uygulaması.

## 🎯 Özellikler

- **4 Ana Ders**: Matematik, Türkçe, Fen Bilimleri, Sosyal Bilgiler
- **Kapsam Seçimi**: 2. Dönem / Tüm Yıl / 5. Sınıfa Hazırlık
- **Ünite Bazlı Seçim**: İstediğin üniteleri seç
- **Soru Sayısı**: 10, 15 veya 20 soru (30 dk hedef)
- **Zorluk Dağılımı**: Rahat / Dengeli / Meydan Okuma
- **Karışık Soru Tipleri**: Çoktan seçmeli + Doğru/Yanlış + Boşluk doldurma
- **Anında Kontrol**: Her sorudan sonra açıklamalı geri bildirim
- **Sonuç Raporu**: Zorluk analizi, yanlış listesi, öğretmen önerisi
- **Danışman Öğretmen**: Haftalık plan, ipuçları, yol haritası
- **Skor Geçmişi**: Tüm test sonuçları kaydedilir
- **Mobil Uyumlu**: Telefon, tablet, bilgisayar
